3. I have incredible abilities. My memory is unlimited

Think about how much data you already have stored in your memory (numbers, stories, jokes, experiences, words, names, and places.) Think about what an incredible memory you need just to have a conversation. You have to listen, create meaning from what you have just heard, and then search your memory for a response. Not even all the computers in existence connected to each other can perform such a feat. You will see your incredible ability once you have learned the memory methods.
